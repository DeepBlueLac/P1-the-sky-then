# 此刻星空产品说明

## 用户任务

为纪念重要时刻的人生成真实星空海报。用户提供日期、当地时间和地点，立即得到可保存或分享的纪念成品。

## 关键词与证据

- 英文：`star map generator`、`night sky map`、`custom star map`、`birthday star map`、`wedding star map`。
- 中文：星图生成器、那天的星空、出生星空图、纪念日星空海报。
- 当前收入和竞品数据仍属于待验证假设，不能据此宣称已验证变现。
- DBL 的视觉设计能力是主要差异变量，计算真实性是信任底座。

## MVP

1. 输入日期、当地时间和地点。
2. 生成真实可见恒星与 IAU 星座连线。
3. 选择三套设计和三种画幅。
4. Web 直接下载高清 PNG。
5. 输入和题词只保存在当前设备。
6. 免费 PNG 始终可用，不设置登录、支付或下载门槛。

## 非目标

- 不做账户、云同步、社区和复杂权限。
- 不做自由图层编辑器、AI 虚构星空或实体打印订单。
- 不做专业天文观测、行星轨迹和卫星追踪。
- 首轮不接真实支付、订单、付费墙或实体印刷履约。

## 验证

- 100 个独立访客。
- 生成完成率不低于 35%。
- 海报下载率不低于 20%。
- 少于 15% 的完成率或几乎无人保存成品时停止扩展。
