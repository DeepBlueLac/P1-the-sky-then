# 隐私说明

此刻星空不要求注册，不建立用户账户，也不把题词或海报上传到服务器。

- 日期、时间、地点、标题和题词保存在当前设备本地。
- 地点搜索会把用户输入的地点关键词发送到 Open-Meteo 地理编码服务。
- 海报在设备端生成；Web 下载和原生分享均不经过项目服务器。
- Web 会通过 Vercel Analytics 记录生成、下载和印刷级版本兴趣事件；只包含主题与画幅枚举。
- iOS/Android 的同类事件只在设备本地累计，不上传。
- 分析事件不包含日期、时间、地点关键词、经纬度、标题或题词，也不用于建立用户画像。
- 当前不接入广告或真实支付；点击印刷级版本兴趣不会扣款、创建订单或收集付款资料。

删除浏览器站点数据或卸载 App 会清除本地保存内容。
