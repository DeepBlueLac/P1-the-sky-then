# 此刻星空：前端体验升级 PRD

## 项目目标

在不改变真实星图生成、地点搜索、本地保存、免费 PNG 下载和付费意向记录功能的前提下，重写 Web 前端，让 `https://the-sky-then-live.vercel.app/` 成为沉浸、清晰、可访问的星空纪念海报工作台。

## 已确认的设计方向

- 背景：全屏自动播放、循环、静音的星空视频：`https://d8j0ntlcm91z4.cloudfront.net/user_38xzZboKViGWJOttwIXH07lWA1P/hf_20260315_073750_51473149-4350-4920-ae24-c8214286f323.mp4`。
- 布局：桌面 52/48 编辑器/预览分栏；移动端按用户任务顺序纵向呈现。
- 风格：黑白灰为基础、深空蓝仅用于星图语义；半透明液态玻璃面板，复用 `.liquid-glass` / `.liquid-glass-strong`，圆角令牌为 `1rem`。
- 字体：Poppins 用于界面与标题，Source Serif 4 用于少量斜体强调；中文提供可靠后备字体。
- 交互：关键按钮必须有 hover、focus、active 反馈；尊重系统减少动态效果偏好；视频不得妨碍可读性或输入操作。

## 范围与约束

- 保留计算、海报 SVG、地理编码 API、隐私边界、事件名称与 Web/iOS/Android 可导出性。
- 不删除用户数据；不提交密码、Token、密钥或个人隐私；不修改目标外的功能。
- Web 视频仅作为视觉背景；不能加载时必须退化为稳定深色背景。
- 不引入未验证的新运行时依赖。

- [ ] `npm run check`、`npm test`、`npm run web:export` 和 `npm run test:e2e` 全部通过。
- [ ] 页面、`/api/health`、`/api/geocode` 的关键请求保持成功；Paris 查询和 PNG 下载可复现。
- [ ] 桌面与移动端均无横向溢出、遮挡、不可读文字或失效的焦点状态。
- [ ] 视觉审计与 `design-qa.md` 均基于当次截图，且无未处理的 P0/P1/P2 问题。
- [ ] 不遗留调试代码、临时文件或明文密钥；Git diff 仅包含本任务需要的文件。

## Ralph 循环

1. 运行健康检查；仅在独立、干净分支启动。
2. 每轮先修复失败测试与已知体验风险，再做最小范围实现。
3. 每轮验证构建与相关测试；遇到不安全的破坏性操作立即停止并记录。
4. 仅当全部任务与完成标准满足后，输出 `<promise>COMPLETE</promise>`。
