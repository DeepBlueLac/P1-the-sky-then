# 项目目标

把“此刻星空”优化成一个功能完整、无明显缺陷、三端体验一致且具有高级视觉质感的真实星图纪念海报工具，并加入诚实、可测量的印刷级数字海报付费意愿验证入口。

## 产品边界

- 产品仍然只解决一个任务：输入日期、当地时间和地点，生成并导出那一刻的真实星空纪念海报。
- 保持 Expo Router + React Native Web 的 Web、iOS、Android 共享架构。
- 保持无强制登录、无复杂权限、纯前端或无状态轻后端；不增加账户、社区、云同步、后台管理或实体印刷订单。
- 免费 PNG 导出必须继续可用，不能用付费实验破坏现有核心功能。
- “付费意愿”只能通过真实可追踪的意向点击验证，不能宣称已有收入、订单或付费用户。
- 允许使用轻量、免费、官方维护的依赖；新增依赖必须解决明确问题并兼容三端。

## 不允许做的事

- 不删除用户数据或无故清除本地保存的海报设置。
- 不提交 `.env.local`、密码、Token、密钥、签名文件、服务账号或个人隐私。
- 不发送日期、地点、题词正文、精确坐标等用户输入到分析服务。
- 不改变真实星位计算、现有 GitHub/Vercel/EAS 架构及目标以外的功能。
- 不制作虚假支付、虚假倒计时、虚假稀缺、默认勾选或误导用户已经购买的界面。
- 不在页面加入营销式大 Hero、装饰性渐变球、卡片套卡片、过度圆角或影响海报查看的动画。
- 遇到无法安全判断的破坏性操作时停止，并说明原因。
